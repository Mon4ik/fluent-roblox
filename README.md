## Installation

### Userstyles.world

- https://userstyles.world/style/16043/fluent-roblox

### Manual

- Create custom style (with any userstyle extension)
- Place this code for domain `www.roblox.com`
  ```css
  @import url("https://fluent-roblox.idkncc.ru/main.css?v=1.0.1");
  ```
